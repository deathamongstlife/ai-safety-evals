# AI Safety Evals

Evaluation prompts and starter datasets for studying reliability issues in language-model systems.

## Focus
- false completion
- instruction conflict
- misleading context
- tool-use restraint
- goal drift

## Structure
- `prompts/` — evaluation prompts
- `datasets/` — starter datasets
- `results/` — pilot observations
- `docs/` — methodology notes
- `scripts/` — helper scripts
